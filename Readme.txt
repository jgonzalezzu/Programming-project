1. Compile "SFF.cpp" in 'single_file' folder using:$ g++ -std=c++11 SSF.cpp

[OPTIONAL] For compile modularized use all ".cpp" files and "constants.h" in 'modularized' folder  importing this files in new Visual Studio Solution.

2. run

2. For plotting the data, go to gnuplot and use:
2.1 for function p(t) :$ load 'plot_p'
2.2 for function Sigma_s :$ load 'plot_Ss'
2.3 for function Sigma_s :$ load 'plot_Sd'
2.4 for concentration :$ load 'plot_concentration.p'
2.5 for emission :$ load 'plot_emissions.p'
2.6 for max value :$ load 'plot_max'

3. Enjoy!
